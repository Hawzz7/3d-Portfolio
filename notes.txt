1. Visit sketchfab websitee for dowloading 3d models for the project you are working. 
2. The model files are .glb files. you need to convert the  files to jsx file to use it in your react project.
3. To do that vist this website "gltf.pmnd.rs".
4. Drag and drop the model file from where ever in your project directory in the website and change the path prefix in the option on the top right corner to "where it is downloaded" and hit enter.
5. After the above step copy the code from the site and paste it inside your project/component
